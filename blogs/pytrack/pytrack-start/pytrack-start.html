Pytrack Blogs

One of my pet projects

  • June, 29 2023

Pytrack was one of my earlier projects and this project is dear to my heart because I usually use it. I made this application to combat my procrastination. Basically, it functions like a parent that will limit the time you spend on unproductive applications and force you to do some work first before you can take a break. I modeled this application based on my behavior, but with some customization in the numbers, you can also make this your own. Feel free to use this application.
