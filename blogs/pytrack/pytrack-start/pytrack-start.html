Pytrack Blogs

One of my pet projects

  • June, 29 2023

Pytrack was one of my earlier project and this project is dear in my heart because I usually use it. I made this application to combat my procrastination. Basically it function like a parent that will limit the time you spent on unproductive application and force you to do some work first before you can take a break. I modeled this application based on my behavior but with some customization in the numbers you can also make this your own. Feel free to use this application.
